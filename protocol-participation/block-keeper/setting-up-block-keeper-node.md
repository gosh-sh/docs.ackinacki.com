# Setting up Block Keeper Node

## **System Requirements**

| Configuration | CPU (cores) | RAM (GiB) | Storage    | Network                                          |
| ------------- | ----------- | --------- | ---------- | ------------------------------------------------ |
| Minimum       | 8c/16t      | 64        | 1 TB NVMe  | 1 Gbit synchronous unmetered Internet connection |
| Recommended   | 12c/24t     | 128       | 2 TB NVMe  | 1 Gbit synchronous unmetered Internet connection |

## How to run

Follow the instructions in [Ackinacki repo.](https://github.com/ackinacki/ackinacki)

## API access

After the Block Keeper is successfully provisioned  the gql endpoint will be on `localhost:3000/graphql`
